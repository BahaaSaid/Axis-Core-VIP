# Axis-Core-VIP
🎫 VIP Generator

An interactive web application that allows users to generate a stylish VIP card dynamically using form inputs.

---

🚀 Features

- Generate a VIP card instantly using DOM manipulation
- Support for user image via URL
- Select membership type (Silver / Gold / Platinum)
- Smooth animations when the card appears
- Hover effects for buttons and card
- Copy button inside the card using the Clipboard API

---

🛠️ Technologies Used

- HTML5
- CSS3 (Animations & Hover Effects)
- JavaScript (DOM Manipulation & Clipboard API)

---

⚙️ How It Works

1. Enter your name
2. Add an image URL (optional)
3. Choose membership type
4. Click Generate
5. The VIP card will appear instantly
6. Click Copy Code to copy the card HTML
